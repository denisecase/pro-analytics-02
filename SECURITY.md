# SECURITY.md

This repository contains public educational documentation and
supporting professional analytics project files.

## Reporting security issues

Do not report security issues by opening a public issue if the report includes:

- credentials,
- tokens,
- private URLs,
- student information,
- unpublished course information,
- sensitive logs,
- security exploit details.

Instead, contact the repository maintainer directly.

## What to report

Please report:

- exposed credentials or secrets,
- unsafe setup instructions,
- dependency or workflow risks,
- instructions that could cause data loss,
- links to malicious or compromised resources,
- security concerns in automation or CI configuration.

## Examples

This repository may include examples, logs, datasets,
screenshots, and project files.

Do not add or expose any private data, real credentials,
protected personal information,
or sensitive operational details.

## AI-assisted work

AI tools may help draft or review changes,
but they must not be used to publish
private data, credentials, or sensitive security details.

Human review remains required for security-relevant changes.
