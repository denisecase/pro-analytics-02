# 🔵 Start in GitHub to Copy a Template Repository

1. Log in to GitHub.

2. In your browser, navigate to the GitHub page of the example project repository you want to copy.

3. Click the green **Use this template** button and select **Create a new repository**.

4. Ensure the **Owner** is your GitHub account.

5. Keep the repository name. It should indicate the associated skills and should not contain spaces.

6. Select **Public** so others can view your repository. You may use a fake name or alias if you prefer.

7. Click **Create repository** to complete the process.
